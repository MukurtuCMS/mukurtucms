-- SUMMARY --

Supports theme condition checks via context


-- REQUIREMENTS --

* Context-3.x - http://drupal.org/project/context
* Chaos Tool Suite - http://drupal.org/project/ctools

-- INSTALLATION --

* Enable module in admin/build/modules


-- CONFIGURATION --

* The module adds a "THEM" condition in the Context UI.
