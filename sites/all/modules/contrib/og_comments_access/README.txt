Organic groups comments access module
=============================

Introduction
---------------

Allow you to manage comment permission on OG group context.

It comes with news OG permissions :

* Administer OG comments
* Modify any OG comments
* Delete own OG comments
* Post OG comments

With these permissions, you can now restrict comment posting on group.
You can also allow particular OG role to modify all group comments.

These permissions overwrite the global Drupal permission in the group context.

Requirements
------------
This module requires the following modules:
 * OG (https://drupal.org/project/og)

Installation:
------------
* Install as you would normally install a contributed drupal module

Configuration & Workflow
------------

* You can define the default OG permission for your groups at :
admin > config >group > permissions
